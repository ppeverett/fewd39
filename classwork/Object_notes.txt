selecting for things on document object:
1. document.querySelector("");

creating new elements on a document:
2. document.createElement("");

=====================

On the element object, we can set the textContent (example: element object is listitem)
3. listitem.textContent = "set it to something"

(example: element object is "image"). Now set the Attribute src = "https..."

4. image.setAttribute("src", "https...");

Now set the attribute of the listItem to class = "green"
5a. listItem.className =  "green";
5b. listItem.setAttribute("class", "green");

=====================
To append a child element to a parent element:
1. select for the parent element: document.querySelector("parent");
2. append the child to the parent using: parent.appendChild(listItem);
